# Semantic Contradiction Ledger

An AI-powered system for detecting semantic contradictions between statements using Natural Language Processing (NLP) and maintaining a blockchain-style immutable ledger for transparent and tamper-proof record tracking.

## 🚀 Features

- Semantic contradiction detection
- NLP-based text analysis
- Contradiction scoring with explanation
- Blockchain-inspired immutable ledger
- Real-time statement comparison
- Clean and lightweight web interface

## 🧠 Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Python Flask
- AI/NLP: Sentence Embeddings, Cosine Similarity
- Storage: JSON-based Blockchain Ledger

## 📌 Example

### Input

Statement 1:
"Company profits increased this year."

Statement 2:
"Company profits decreased this year."

### Output

Contradiction Score: 92%

Reason:
Opposite semantic meaning detected between statements.

## 📖 Research Motivation

Most existing systems focus on plagiarism detection, semantic similarity, or fake news classification. This project focuses on semantic contradiction detection combined with immutable ledger storage for improved transparency, accountability, and traceability.
